# Image Puzzle Game

A simple and interactive puzzle game where you can upload any image and solve it by arranging shuffled puzzle pieces.

## Features

- Upload any image from your computer
- Choose from 3 difficulty levels:
  - Easy: 3x3 grid (9 pieces)
  - Medium: 4x4 grid (16 pieces)
  - Hard: 5x5 grid (25 pieces)
- Drag and drop puzzle pieces to arrange them
- Timer to track how fast you solve the puzzle
- Move counter to track efficiency
- Win animation and celebration message
- Responsive design for desktop and mobile devices

## How to Play

1. Open index.html in your web browser
2. Click "Upload Image" and select an image file from your computer
3. View the image preview
4. Select your preferred difficulty level
5. Click "Start Game" to begin
6. Drag puzzle pieces from the "Puzzle Pieces" panel on the right into the empty slots in the middle
7. Arrange all pieces in their correct positions to solve the puzzle
8. Once complete, you will see a congratulations message with your time and move count
9. Click "Play Again" to restart with the same image

## Files

- index.html: Main HTML structure of the game
- styles.css: Complete styling and responsive layout
- script.js: Game logic including image processing, puzzle generation, and drag-drop functionality

## Technical Details

- Built with vanilla HTML5, CSS3, and JavaScript
- Uses Canvas API for image manipulation and puzzle piece generation
- Implements drag and drop API for interactive gameplay
- No external dependencies required

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge
